How to upload to GitHub Pages:
1. Open your GitHub repository.
2. Upload these files directly into the main branch.
3. Go to Settings > Pages.
4. Source: Deploy from branch > main > root.
5. Save and wait a few minutes.

Important:
- Keep index.html, profile.jpg and avatar.jpg in the same folder.
- Replace the LinkedIn placeholder link inside index.html with your real LinkedIn URL.
